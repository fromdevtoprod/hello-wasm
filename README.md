# Hello-wasm

Hello-wasm is a simple test.
The goal was to compile a Rust project into WebAssembly and use it in an existing web app.

https://developer.mozilla.org/en-US/docs/WebAssembly/Rust_to_Wasm

## Usage

To build the project :
```
npm run build
```

To use it locally :
```
npm run start
```
