# Hello-wasm

Hello-wasm is a simple test following this technical documentation reading :

https://developer.mozilla.org/en-US/docs/WebAssembly/Rust_to_Wasm

## Usage

To build the project :
```
npm run build
```

To use it locally :
```
npm run start
```
